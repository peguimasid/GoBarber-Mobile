### Iniciando na sua maquina
1. Inicie o [servidor do GoBarber](https://github.com/peguimasid/GoBarber-backend-pt2)
2. Baixe o projeto
3. `cd gobarberMobile-master`
4. `yarn`
5. `yarn ios` ou `yarn android`
